# 도커
## docker compose
다중 컨테이너 도커 애플리케이션을 정의하고 실행할 수 있도록 도와주는 도구이다. yaml 파일을 사용하여 각종 구성을 설정한다.  
- 멀티 컨테이너 정의
  - 여러 컨테이너를 하나의 앱처럼 정의하고 관리
- 자동화된 실행
  - 단일 명령(docker-compose up)으로 모든 컨테이너 실행
- 환경 설정 분리
  - 개발, 테스트, 프로덕션 환경별로 설정을 관리
- 효율적인 관리
  - 하나의 파일(docker-compose.yml)로 모든 의존성과 실행 구성을 관리

> compose: 구성하다, 조립하다
